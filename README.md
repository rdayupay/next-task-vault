# Task Vault

Task Vault is a project management tool designed to simplify task tracking and issue resolution.

## Technologies Used

- Next.js
- MongoDB
- Mongoose
- React
- Tailwind CSS
- ESLint

## Getting Started

First, install the dependencies:

```bash
$ npm install
```

Then, start a local development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
